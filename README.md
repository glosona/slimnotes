# SlimNotes

**macOS용 초경량 마크다운 메모 앱**

> 가볍게 빠르게 마크다운 메모를 할 수 있는 앱입니다.

![SlimNotes Screenshot](./Slim%20Notes.png)

---

## AI 활용

Swift를 한 번도 써본 적 없는 상태에서 시작했습니다. Claude AI에게 앱 구상도를 전달해 초기 코드를 생성하고, 그 코드를 읽으며 Swift/SwiftUI 문법을 학습했습니다.

단순히 AI가 짜준 코드를 복붙한 게 아니라, **직접 코드를 읽고 → 문제를 파악하고 → 요구사항을 구체화해서 전달하는** 과정을 반복했습니다.

| 요구사항 | 내용 |
|---|---|
| **스타일 커스터마이징** | native 버튼 대신 custom 버튼으로 교체, 투명도 조절 슬라이더 추가 등 |
| **마크다운 렌더링 디버깅** | `LiveMarkdownEditor.swift`를 직접 분석하며 버그 원인 파악 후 수정 요구사항 전달 |
| **성능 문제 해결** | SwiftUI 실시간 렌더링의 렉 → Enter 입력 시 적용되는 방식으로 변경하여 해결 |
| **경량화 판단** | 외부 라이브러리 없이 AppKit(`NSTextView`, `NSTextStorage`) 직접 사용 |

## 주요 기능

- 실시간 마크다운 렌더링
- 헤더/강조 마크다운 기호는 렌더링 후 숨김 처리, 리스트 기호는 유지
- 앱 창 배경 투명도 조절
- 메모 자동 저장 (JSON 기반)
- 사이드바 메모 관리 (생성/삭제/선택)

### 지원 마크다운 문법

| 문법 | 예시 | 비고 |
|---|---|---|
| 제목 | `# H1` `## H2` `### H3` | 기호 숨김 + 크기/굵기 적용 |
| 강조 | `**굵게**` `_기울임_` `~~취소선~~` | 기호 숨김 |
| 인라인 코드 | `` `code` `` | 배경 하이라이트 |
| 목록 | `- 항목` `1. 항목` | 기호 유지 (dim 처리) |
| 코드 블록 | ` ``` ` | 모노스페이스 폰트 |
| 체크리스트 | `- [ ]` `- [x]` | ⚠️ 동작 불안정 — 수정 예정 |
| 수평선 | `---` `===` `___` | ⚠️ 동작 불안정 — 수정 예정 |

## 설치 및 실행

### 방법 1 — 앱 직접 다운로드 (권장)

1. [Releases](https://github.com/glosona/SlimNotes/releases/latest)에서 `SlimNotes.app.zip` 다운로드
2. 압축 해제 후 `/Applications` 폴더로 이동
3. 처음 실행 시 우클릭 → **열기** (macOS 보안 경고 우회)

> Apple 개발자 서명이 없기 때문에 첫 실행 시 Gatekeeper 경고가 뜹니다.
> 시스템 설정 → 개인 정보 보호 및 보안 → "확인 없이 열기"로도 허용 가능합니다.

### 방법 2 — 소스에서 빌드

```bash
git clone https://github.com/glosona/SlimNotes.git
cd SlimNotes
open SlimNotes.xcodeproj
# Xcode에서 ⌘+R로 빌드 및 실행
```

**요구 사항**
- macOS 13 이상
- Xcode 15 이상

## 기술 스택

| 영역 | 기술 |
|---|---|
| UI 프레임워크 | **SwiftUI** |
| 텍스트 편집 | **NSTextView** (AppKit) — `NSViewRepresentable`로 브리지 |
| 마크다운 렌더링 | **NSTextStorage + NSLayoutManager** — 속성 기반 실시간 렌더링 |
| 데이터 저장 | **JSON (Codable)** — 파일시스템 직접 저장 |
| 외부 의존성 | **없음** — 순수 Apple 프레임워크만 사용 |
